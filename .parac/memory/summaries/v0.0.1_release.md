## ğŸš€ Key Features Highlights

### 1. Skill System
```python
from paracle_orchestration.coordinator import AgentCoordinator

coordinator = AgentCoordinator(
    enable_skills=True,
    skill_injection_mode="full"
)

result = await coordinator.execute_agent(
    agent=agent_spec,
    inputs={"task": "Implement feature"},
    load_skills=True
)
```

### 2. Phase 5 Sandbox
```python
from paracle_sandbox import SandboxManager, SandboxConfig

config = SandboxConfig(
    image="python:3.10-slim",
    cpu_limit=1.0,
    memory_limit="512m"
)

async with SandboxManager() as manager:
    sandbox = await manager.create_sandbox(config)
    result = await sandbox.execute("python script.py")
```

### 3. API-First Workflow
```bash
# Start API server
paracle serve

# Execute workflow
curl -X POST http://localhost:8000/api/workflows/execute \
  -H "Content-Type: application/json" \
  -d '{"workflow_id": "my-workflow", "inputs": {}}'
```

### 4. Multi-Framework Support
```python
from paracle_adapters import (
    LangChainAdapter,
    LlamaIndexAdapter,
    CrewAIAdapter,
    AutoGenAdapter
)

# Use any framework
adapter = LangChainAdapter()
result = await adapter.execute_agent(agent_spec)
```
