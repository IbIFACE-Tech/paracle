# Google Gemini - Paracle Instructions

Using **PARACLE** with `.parac/` as the single source of truth.

## Core Principle

> **`.parac/` is your single source of truth. Read it. Follow it. Log to it.**

## Source Files

| File | Purpose |
|------|---------|
| `.parac/agents/specs/*.md` | Agent definitions |
| `.parac/memory/context/current_state.yaml` | Project state |
| `.parac/roadmap/roadmap.yaml` | Roadmap and phases |
| `.parac/GOVERNANCE.md` | Governance rules |
| `.parac/memory/logs/agent_actions.log` | Action history |

## Workflow

1. **Read Context**: Load `.parac/memory/context/current_state.yaml`
2. **Select Agent**: Choose from `.parac/agents/specs/`:
   - `pm.md` - Project Management
   - `architect.md` - Architecture
   - `coder.md` - Implementation
   - `tester.md` - Testing
   - `reviewer.md` - Code Review
   - `documenter.md` - Documentation
3. **Read Agent Spec**: Load specification from `.parac/agents/specs/{agent}.md`
4. **Apply Standards**: Follow agent's responsibilities
5. **Execute Task**: Complete the work
6. **Log Action**: Use governance logger:

   ```python
   from paracle_core.governance import log_action, agent_context

   with agent_context("CoderAgent"):
       log_action("IMPLEMENTATION", "Added feature X")
   ```

7. **Update Memory**: Propose updates to `.parac/` files

## Action Types

| Action | Use When |
|--------|----------|
| `IMPLEMENTATION` | Writing new code |
| `TEST` | Creating tests |
| `BUGFIX` | Fixing bugs |
| `REFACTORING` | Improving code |
| `REVIEW` | Code review |
| `DOCUMENTATION` | Writing docs |
| `DECISION` | Architecture decisions |

## Gemini Features

- Leverage long context window for `.parac/` analysis
- Use multimodal capabilities when needed
- Apply multi-step reasoning for complex tasks

## Dogfooding Context

```
packages/   ← Framework code (product)
.parac/     ← Project workspace (dogfooding)
```

## Rules

**DO:**
- Read `.parac/` for context before acting
- Log all significant actions
- Follow agent specifications

**DON'T:**
- Duplicate agent definitions
- Create custom governance rules
- Ignore `.parac/` state

---

**Remember**: `.parac/` is your single source of truth. Always read it first.
